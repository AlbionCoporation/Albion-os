ALBION OS 5.0 — FULL PROJECT DESCRIPTION

Albion OS 5.0 is a web-based operating system simulation built entirely using HTML5, CSS3, and JavaScript. It is designed as a desktop environment that runs inside a browser and mimics the behavior of a real operating system.

This project was created by Albion Corporation as a conceptual operating system UI/UX experiment.

--------------------------------------------
GENERAL OVERVIEW
--------------------------------------------

Albion OS provides a full interactive desktop interface including boot screen, login system, taskbar, start menu, draggable windows, and multiple built-in applications.

It is not a real kernel-based OS but a frontend simulation designed for educational and experimental purposes.

--------------------------------------------
CORE FEATURES
--------------------------------------------

- Boot screen with animated startup sequence
- Login screen with basic authentication UI
- Desktop environment with icons
- Taskbar with clock system
- Start menu for quick app access
- Window manager system (open, close, minimize, maximize)
- Z-index focus system for active windows
- Drag and move window functionality

--------------------------------------------
APPLICATIONS INCLUDED
--------------------------------------------

Albion OS includes multiple built-in applications:

1. Notes Application
- Simple text editor
- Auto-save using localStorage
- Persistent data storage in browser

2. Calculator
- Basic arithmetic operations
- Expression evaluation system

3. Browser Simulation
- URL input field
- iframe-based web rendering
- fallback search handling

4. File Explorer
- File upload system
- Supports:
  - .txt files
  - .png images
  - .jpg images
  - .mp3 audio files
  - .mp4 video files
- Dynamic preview system

5. Paint Application
- Canvas-based drawing tool
- Mouse drag drawing system
- Color picker
- Brush size control
- Clear canvas function

6. Terminal Emulator
- Command-line interface simulation
- Commands:
  - help
  - time
  - ver
- Real-time output logging

7. Settings Panel
- Wallpaper customization
- Version display
- License information
- System configuration options

8. Tic Tac Toe Game
- 3x3 grid game system
- Two-player alternating turns
- Reset functionality

--------------------------------------------
WINDOW SYSTEM
--------------------------------------------

Albion OS uses a custom window manager system that allows:

- Opening multiple windows
- Closing windows individually
- Minimizing windows
- Maximizing and restoring windows
- Layer management (z-index control)
- Focused window highlighting

--------------------------------------------
USER INTERACTION SYSTEM
--------------------------------------------

The system is fully interactive using JavaScript event listeners:

- Click events for opening apps
- Keyboard input for terminal and shortcuts
- Mouse drag events for window movement
- File input events for file explorer
- Timer-based system updates (clock, boot animation)

--------------------------------------------
DESIGN PHILOSOPHY
--------------------------------------------

Albion OS is designed with a dark modern UI theme inspired by:
- Modern Linux distributions
- Windows-like desktop environments
- Minimalist cyber UI aesthetics

The focus is on simplicity, usability, and visual clarity.

--------------------------------------------
TECHNICAL STRUCTURE
--------------------------------------------

Built using:

- HTML5 for structure
- CSS3 for styling and layout
- Vanilla JavaScript for logic and interactivity

No external frameworks or libraries are required.

--------------------------------------------
STORAGE SYSTEM
--------------------------------------------

Uses browser localStorage for:
- Saving notes
- Persisting user text data

--------------------------------------------
LIMITATIONS
--------------------------------------------

- Not a real operating system kernel
- Runs entirely inside a browser
- No real file system access
- Limited performance for heavy operations
- Browser dependency required
